## Install Nexys with npm

```bash
npm install nexys
```

## Import Nexys to your project

```javascript
import Nexys from "nexys";
```

## Then initialize Nexys

```javascript
const nexys = new Nexys("API_KEY", { appName: "APP_NAME" });
```

## Start logging!

```javascript
nexys.log("Hello World!");
```

## Or give an syntetic error

```javascript
nexys.error("Probably some failure");
```

## What about real error?

```javascript
throw new Error("My excellent error");
```

- Errors will be caught and will be sent to dashboard by Nexys automatically if specified in options.

## Or complex objects

```javascript
nexys.log({
  test: true,
  myBigObject: {
    okay: false,
    iUseNexys: true
  }
});
```
